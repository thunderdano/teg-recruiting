# Build With Us — TEG Adventure Factory recruiting (POC)

Static landing page (single `index.html` + `img/`). No build step.

## Deploy (GitHub Pages, like `signal`)
Pushing to `main` auto-deploys via `.github/workflows/deploy.yml`.
In the repo: **Settings → Pages → Build and deployment → Source: GitHub Actions** (one-time).

- Apply form is a demo (logs the lead to the browser console; not wired to an ATS yet).
